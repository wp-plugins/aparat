نمایش ویدئوی آپارات در مطالب توسط شورت‌کد ...

== Description ==

با فعالسازی این افزونه می‌توانید ویدئوی مورد نظر در آپارات را در مطلب ارسالی قرار دهید ...

* شورت‌‌کد :
`[aparat id="c4VeZ"]`

* شورت‌کد با امکان شخصی‌سازی :
`[aparat id="c4VeZ" width="500" height="500" style="margin: 15px; padding: 7px"]`

== Installation ==

1. به برگه 'افزونه‌ها' در مدیریت وردپرس رجوع کرده و بر روی 'افزودن' کلیک کنید
2. 'Aparat' را جستجو کنید
3. افزونه ذکر شده را نصب کرده و توضیحات صفحه‌ی مدیریت آن را مطالعه کنید
